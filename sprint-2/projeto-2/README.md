# Projeto 2: Biblioteca Triple Peaks

A página da biblioteca Triple Peaks é o segundo projeto no programa de desenvolvimento web na TripleTen. Ela foi criada usando HTML e CSS, com base no roteiro.

## Descrição

A página apresenta uma biblioteca virtual com:
- **Header**: Seção welcoming com navegação, título de boas-vindas, descrição e imagem ilustrativa
- **Navegação**: Menu com links para Escolhas da Equipe, Eventos, Seja um membro e Contatos
- **Seção Principal**: Apresentação de "Escolhas da Equipe" com 4 cards de livros em grid de 2 colunas
- **Cards**: Cada card contém informações sobre um livro (título, descrição e autores)

## Recursos do projeto

- HTML5 semântico
- Flexbox para layouts responsivos
- CSS Grid para arranjo de cards
- Background images
- Tipografia: Segoe UI
- Responsive design com viewport meta tag
- Normalize.css para consistência entre navegadores
